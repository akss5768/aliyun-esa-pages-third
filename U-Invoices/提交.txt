# U-Invoices 发票管理

U-Invoices 是一个现代化的发票管理应用，帮助您轻松管理相关信息。

## 功能特性

- **发票录入**
- **分类管理**
- **统计分析**
- **导出功能**
- **数据持久化** - 使用 LocalStorage 本地存储数据
- **响应式设计** - 完美适配桌面和移动设备
- **明亮清新** - indigo主题界面

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS
- **状态管理**: React Hooks
- **数据存储**: LocalStorage
- **构建工具**: Vite
- **图标库**: Lucide React
