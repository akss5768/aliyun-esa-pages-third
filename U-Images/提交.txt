# U-Images 图片画廊

U-Images 是一个现代化的图片画廊应用，帮助您轻松管理相关信息。

## 功能特性

- **图片展示**
- **分类管理**
- **瀑布流布局**
- **放大查看**
- **数据持久化** - 使用 LocalStorage 本地存储数据
- **响应式设计** - 完美适配桌面和移动设备
- **明亮清新** - pink主题界面

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS
- **状态管理**: React Hooks
- **数据存储**: LocalStorage
- **构建工具**: Vite
- **图标库**: Lucide React
